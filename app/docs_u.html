Вступ

Ця документація охоплює використання застосунку ImgProPlus, детально описуючи налаштування та використання застосунку для генерації PROMPTs. Застосунок використовує моделі CLIP для генерації запитів на основі введених зображень, пропонуючи різні моделі для різних випадків використання.

Обробка зображення

Для обробки зображення в застосунку ImgProPlus слід виконати наступні кроки:

  1. Клацніть кнопку "Завантажити зображення".
  2. Виберіть файл зображення, який потрібно обробити, з вашого комп'ютера.
  3. Після завантаження зображення воно буде відображено в області полотна.
  4. За необхідності відрегулюйте розмір зображення, розтягуючи або заповнюючи його за допомогою наданих елементів управління.
  5. У розділі "Налаштування" виберіть бажані параметри для пристрою, моделі, режиму та обчислення.
  6. За необхідності клацніть кнопку "Додаткові налаштування", щоб налаштувати додаткові параметри.
  7. Клацніть кнопку "Обробити зображення", щоб розпочати обробку.
  8. Дочекайтеся завершення обробки. Анімація завантаження вказуватиме на те, що обробка триває.
  9. Після завершення обробки результати будуть відображені у новому вікні.
  10. Огляньте результати та проаналізуйте згенеровані запити на основі вибраних налаштувань.
  11. Ви можете повторити процес з іншими зображеннями або налаштуваннями за потреби.

Встановлення нових моделей

Щоб встановити нову модель, виконайте наступні кроки:

  1. Відкрийте розділ "Моделі".
  2. Виберіть потрібну або бажану модель.
  3. Клацніть "Встановити". Застосунок завантажить вказану модель для вас.
  4. У разі відсутності преміум-підписки розгляньте можливість її придбання для завантаження всіх кращих моделей.

Придбання преміум

Щоб придбати преміум для вашого облікового запису, виконайте наступні дії:

  1. Відкрийте розділ "Обліковий запис".
  2. Клацніть кнопку "Придбати преміум".
  3. Дотримуйтесь інструкцій у новому вікні для завершення покупки.
  4. Після успішного завершення покупки ви отримаєте інформацію про преміум у вашому обліковому записі.
  5. Якщо ваші моделі не доступні для завантаження, спробуйте перезапустити застосунок.

Параметри конфігурації

Кліп-перекладач може бути налаштований за допомогою різних параметрів конфігурації. Нижче подано докладний опис кожного параметра:

Параметр Опис
device Вказує пристрій для обчислень. Опції включають cpu та gpu (для CUDA).
clip_model Модель CLIP для використання. Приклади включають ViT-B/32, ViT-L/14 тощо.
caption_max_length Встановлює максимальну довжину згенерованих підписів. Більше значення створює більш довгі, детальніші підписи.
chunk_size Визначає розмір фрагменту для обробки. Налаштування цього значення може вплинути на продуктивність та використання пам'яті.
flavor_intermediate_count Вказує кількість проміжних варіантів для врахування. Корисно для генерації більш нюансованих запитів.
blip_image_eval_size Встановлює розмір оцінки зображення у налаштуваннях Blip. Цей параметр визначає розміри зображення, що використовується для оцінки під час обробки.
blip_max_length Встановлює максимальну довжину запитів у налаштуваннях Blip. Більше значення створює більш довгі, детальніші запити.
blip_offload Визначає, чи увімкнено вивантаження в налаштуваннях Blip. Якщо встановлено значення True, деякі завдання можуть бути вивантажені для поліпшення продуктивності.
caption_model_name Вказує назву моделі підпису для використання. Цей параметр визначає конкретну архітектуру моделі, що використовується для генерації підписів.
mode Визначає режим обробки для використання. Доступні опції включають fast, negative, classic та best. Кожен режим пропонує різні компроміси між швидкістю та точністю.
compute Вказує платформу обчислень для використання. Опції включають local для локальної обробки та cloud для хмарної обробки.
